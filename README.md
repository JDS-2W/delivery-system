Um sistema prático e eficiente focado no gerenciamento de pedidos e controle de delivery para restaurantes e lanchonetes. A aplicação permite que os estabelecimentos gerenciem seu cardápio e controlem o fluxo de entregas de forma fluida.

## 🚀 Funcionalidades Principais

- **🛒 Gestão de Pedidos:** Controle completo do ciclo de vida dos pedidos.
- **🍔 Catálogo de Produtos:** Cadastro dinâmico de categorias e produtos disponíveis no cardápio.
- **🧾 Emissão de Recibos:** Geração de recibos detalhados para os clientes.
- **⚙️ Painel Administrativo:** Área dedicada para os donos gerenciarem o fluxo do negócio.

## 💻 Tecnologias Utilizadas

**Backend:**
- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server

**Frontend:**
- HTML5, CSS3 e JavaScript puro (Arquivos estáticos servidos via `wwwroot`)
